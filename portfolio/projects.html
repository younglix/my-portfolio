Rider Order Workflow Demo

A simulated rider delivery app built with HTML, CSS, and JavaScript. It demonstrates a complete order lifecycle with sequential step updates, timestamps, and persistent order history. Designed to mimic backend workflow logic.

  • Sequential order state machine (Accept → Vendor → Pickup → Customer → Confirm)
  • Step-level timestamps for audit/logging
  • Persistent order history using localStorage
  • Single active order constraint
  • Interactive workflow simulation
